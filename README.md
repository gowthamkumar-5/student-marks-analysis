# Student Performance Prediction System

## Description
This project predicts whether a student will pass or fail based on marks, attendance, and study hours.

## Features
- Data Cleaning
- Feature Engineering
- Data Visualization
- Machine Learning Model (Logistic Regression)
- Prediction System

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Output
- Top students identified
- Weak students identified
- Model accuracy displayed
